HTML Tags

<p>

The paragraph tag is used to define a paragraph of text.

<p>This is a paragraph</p>

<head>

The head tag contains meta-information about the HTML document.

<head>
    <title>Document Title</title>
    <meta charset="UTF-8">
    <meta name="description" content="Description of the document">
</head>

<body>

The body tag contains the content of the HTML document.

<body>
    <h1>Heading</h1>
    <p>Paragraph</p>
</body>

<div>

The div tag is a generic container for grouping elements.

<div>
    <p>Paragraph 1</p>
    <p>Paragraph 2</p>
</div>

<span>

The span tag is an inline container used for styling a specific part of text.

<p>This is a <span style="color: red;">red</span> word.</p>

<h1>-<h6>

The heading tags define headings of different levels.

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

<strong>

The strong tag is used to highlight important text.

<p>This is <strong>important</strong>.</p>

<em>

The em tag is used to emphasize text.

<p>This is <em>emphasized</em>.</p>

<br>

The br tag inserts a line break.

<p>Line 1<br>Line 2</p>

<hr>

The hr tag inserts a horizontal line.

<p>Above the line</p>
<hr>
<p>Below the line</p>

CSS Properties

color

Sets the color of text.

color: blue;

font-size

Sets the size of the font.

font-size: 16px;

background-color

Sets the background color of an element.

background-color: #ff0000;

margin

Sets the margin of an element.

margin: 10px;

padding

Sets the padding of an element.

padding: 10px;

border

Sets the border of an element.

border: 1px solid #000;

text-align

Aligns the text within an element.

text-align: center;

font-family

Sets the font family for text.

font-family: Arial, sans-serif;
